# Portfolio John Doe - Développeur Web

## À propos du projet

C’est un site portfolio pour John Doe, un développeur web fictif. Il présente ses compétences, services et projets. Le site est responsive et créé avec React.js et Bootstrap 5.

## Prérequis

Avant de commencer, assurez-vous d’avoir installé sur votre ordinateur :

Node.js (version 16 ou supérieure)

npm (version 8 ou supérieure) ou yarn

Git

Vérifiez vos versions avec :

- node --version
- npm --version
- git --version

## Installation

Cloner le dépôt :

- git clone https://github.com/votre-username/evaluation-reactjs.git
cd evaluation-reactjs

Installer les dépendances :

npm install

## Lancement

Pour démarrer le projet en développement :

- npm start


Le site sera disponible sur http://localhost:3000
